# PacMan

## Comment l'éxécuter
Utilisez la commande suivante dans le dossier.
### Linux

```bash
./run_linux.sh
```
### Windows

Vous pouvez le lancer directement en cliquant sur le .bat

## Commandes de base

### Commandes Pacman:
* Monter: Z
* Gauche: Q
* Descendre: S
* Droite: D

### Commandes Fantôme:
* Monter: ↑
* Gauche: ←
* Descendre: ↓
* Droite: →

Toutes ces commandes sont modifiables dans les paramètres.
